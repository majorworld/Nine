按Alt+1至9：切换到指定桌面

按Ctrl+1至9：活动窗口移到指定桌面
### 拓展
按Alt+0：隐藏或显示任务栏

按Ctrl+0：打开活动窗口文件位置

按Alt+Home/End：置顶或取消置顶窗口

按Alt+PgUp/PgDn：提高或降低非透明度
